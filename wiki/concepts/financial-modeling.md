# Financial Modeling

## Definition

Financial modeling is the process of building mathematical representations of a company's financial performance using historical data, calculated ratios, forecast assumptions, and linked formulas. Models are built in spreadsheet software (typically Excel) to project future financial performance and support investment and business decisions.

## Explanation

A financial model serves as the foundation for any decision involving the projection of a company's future potential. It allows analysts to measure a company's historic performance and create forecasts for future performance, enabling better-informed investment decisions.

Unlike basic spreadsheets, financial models include:
- **Inputs:** Historical data, assumptions, key drivers
- **Outputs:** Forecasted financial statements, valuations, metrics
- **Calculations:** Formulas linking inputs to outputs
- **Scenarios:** Multiple cases (base, bull, bear) for sensitivity analysis

The key skill in financial modeling is creating sensible, justifiable assumptions that fill gaps in the data while maintaining model integrity.

## The 10 Model Types

Financial modeling encompasses many model types, each suited to different purposes:

1. [[three-statement-model]] — Income Statement, Balance Sheet, Cash Flow (the foundation)
2. Forecast Models — Division/company performance analysis
3. [[lbo-model]] — Leveraged buyout structuring
4. [[dcf-model]] — Discounted Cash Flow valuation
5. [[ma-model]] — Merger and acquisition impact analysis
6. Sum-of-the-Parts Model — Valuing separate divisions
7. Comparable Company Analysis — Relative valuation using peer multiples
8. IPO Model — Financial projections for companies going public
9. Divestiture Modeling — Separating divested business financials
10. Option Pricing Model — Theoretical value of derivatives
